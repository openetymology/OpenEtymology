# OpenEtymology 示例 App

这是一个 SwiftUI 示例 App，用来展示 OpenEtymology 的阅读、搜索和练习流程。

包含：

- 500 词 EN-CN 演示 SQLite 数据库
- 500 词 EN-EN 演示 SQLite 数据库
- 公开考试词表文本文件
- 仅用于演示的 StoreKit stub，不包含真实商品 ID

不包含：

- 完整 50,000+ 词生产词典数据库
- App Store 生产配置
- 真实购买流程或商业权益逻辑

使用方式：

用 Xcode 打开 `OpenEtymologySample.xcodeproj`，运行 `wordety` scheme。
